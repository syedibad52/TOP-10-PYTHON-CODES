# 🚀 Top 10 Python Programs (With Examples)

📌 Overview

This repository contains 10 essential Python programs focused on building strong fundamentals in programming.
Each program demonstrates core logic-building techniques used in coding interviews and academic learning.

⚡ Clean, simple, and easy-to-understand implementations for beginners.

This repository contains **basic Python programs** to understand core logic and number-based problems.

---

## 📂 Program Details

| No. | Program                  | Explanation                                               | Example                |
| --- | ------------------------ | --------------------------------------------------------- | ---------------------- |
| 1️⃣ | Armstrong Number         | Checks if sum of digits raised to power equals the number | 153 → 1³+5³+3³ = 153 ✅ |
| 2️⃣ | Counting Digits          | Counts total digits in a number                           | 12345 → 5 digits       |
| 3️⃣ | Even or Odd              | Checks if number is divisible by 2                        | 10 → Even, 7 → Odd     |
| 4️⃣ | Multiplication of Digits | Multiplies all digits of a number                         | 123 → 1×2×3 = 6        |
| 5️⃣ | Palindrome               | Checks if number reads same forward & backward            | 121 → Palindrome       |
| 6️⃣ | Reverse Number           | Reverses the digits of a number                           | 123 → 321              |
| 7️⃣ | Sum of Squares           | Adds squares of each digit                                | 12 → 1²+2² = 5         |
| 8️⃣ | Strong Number            | Sum of factorial of digits equals number                  | 145 → 1!+4!+5! = 145   |
| 9️⃣ | Addition of Digits       | Adds all digits of a number                               | 123 → 1+2+3 = 6        |
| 🔟  | Happy / Sad Number       | Repeated square sum ends in 1 (Happy) or loop (Sad)       | 19 → Happy             |
